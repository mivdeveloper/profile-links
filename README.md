# 🌐 Perfil de Links Sociais

Este é um site pessoal simples e responsivo para reunir seus principais links sociais. Ideal para usar como bio no Instagram, GitHub ou para apresentar seu portfólio de forma rápida e direta.

> 💡 Este projeto foi **inspirado em um desafio do Frontend Mentor**, plataforma que oferece projetos reais para praticar HTML, CSS e JavaScript.

---

### 💡 Funcionalidades

- ✅ Design limpo e moderno
- ✅ Layout 100% responsivo (mobile-first)
- ✅ Foto de perfil com efeito de zoom
- ✅ Botões com bordas arredondadas e efeito hover
- ✅ Rodapé fixado ao final da tela

---

### 🚀 Tecnologias utilizadas

- HTML5
- CSS3
